## ArcGIS API 접근 정보

```
베이스 URL: https://urban.seoul.go.kr/proxy/proxy.jsp?http://98.33.2.225:6080/arcgis/rest/services/UPIS/20200526_WMS/MapServer

쿼리 예시:
  전체 조회: {base}/{layerId}/query?where=1%3D1&outFields=*&returnGeometry=true&outSR=4326&f=json
  이름 검색: {base}/{layerId}/query?where=DGM_NM+LIKE+'%25{검색어}%25'&outFields=*&returnGeometry=true&outSR=4326&f=json
  건수만:    {base}/{layerId}/query?where=1%3D1&returnCountOnly=true&f=json

주요 필드: OBJECTID, PRESENT_SN(공식코드), DGM_NM(구역명), DGM_AR(면적), SIGNGU_SE(시군구코드), PROPEL_CD(진행단계코드)

PROPEL_CD 코드 체계:
  PP{type_prefix}{stage_suffix}

  1) 정비사업 (BZ101~BZ108, BZ301~BZ306) — suffix 기반:
     prefix: 01(BZ101 재개발), 02(BZ102~105), 03(BZ107 가로주택),
             08(BZ301 재정비촉진), 18(BZ302 뉴타운재개발) 등
     suffix: 01=추진중(BZ101전용, 상세불명→빈값), 03=완료/준공(BZ101전용)
             04=구역지정, 05=추진위, 06=조합설립, 07=사업시행, 08=관리처분,
             09=착공, 10=준공, 11=이전고시(준공), 12=조합해산

  2) 지구단위계획 (BZ202, Layer 102) — PP05xx:
     PP0500~PP0506 = 모두 "결정" (결정고시 완료 상태)
     ArcGIS에 폴리곤이 존재 = 이미 결정고시됨. 세부 분류(변경차수 등)는 불명확

  3) 도시재생 BZ401 (Layer 112) — PP10xx:
     PP1001=지정 (촉진지구 지정), PP1002=해제 (촉진지구 해제)

  4) 도시재생 BZ402 (Layer 113) — PP11xx:
     suffix는 정비사업과 동일 (04~12) + 13=해제

  5) 도시재생 BZ403/BZ404 (Layer 114~115) — PP21xx:
     PP2101=지정, PP2102=진행, PP2103=해제
```
