# 정비사업 유형 분류 체계 (법령 기반)

> PropValue DB `project_type` 필드의 기준 문서
> 작성: 2026-05-06 / 법령 기준: 2025~2026년 현행법

---

## 1. 법령별 사업유형 트리

```
┌─ ① 도시 및 주거환경정비법 (도시정비법)
│   ├─ 주거환경개선사업
│   ├─ 재개발사업
│   │   ├─ 주택정비형 재개발  ← 구 "주택재개발"
│   │   └─ 도시정비형 재개발  ← 구 "도시환경정비/상업지역 재개발"
│   └─ 재건축사업
│
├─ ② 빈집 및 소규모주택 정비에 관한 특례법 (소규모주택정비법)
│   ├─ 자율주택정비사업
│   ├─ 가로주택정비사업
│   ├─ 소규모재건축사업
│   └─ 소규모재개발사업      ← 2021.4 신설 (역세권/준공업지역)
│
├─ ③ 도시재정비 촉진을 위한 특별법 (재정비촉진특별법)
│   └─ 재정비촉진사업 (= 뉴타운)
│       ├─ 주거지형
│       ├─ 중심지형
│       └─ 고밀복합형
│       (내부 사업: ①②의 개별사업 + 도시개발사업 등을 포함)
│
├─ ④ 도시재생 활성화 및 지원에 관한 특별법 (도시재생특별법)
│   └─ 도시재생활성화사업
│       ├─ 우리동네살리기형
│       ├─ 주거정비지원형
│       ├─ 일반근린형
│       ├─ 중심시가지형
│       └─ 경제기반형
│
└─ ⑤ 기타 개별법
    ├─ 택지개발촉진법 → 택지개발사업 (2015 폐지, 잔여사업만)
    ├─ 도시개발법 → 도시개발사업
    ├─ 국토계획법 → 지구단위계획, 도시계획시설
    ├─ 혁신도시법 → 혁신도시
    ├─ 기업도시개발특별법 → 기업도시
    ├─ 경제자유구역법 → 경제자유구역
    ├─ 신행정수도특별법 등 → 신도시
    └─ 주택법 → 리모델링
```

## 2. 서울시 고유 브랜드 (법적 사업유형 아님, 행정 프로그램)

| 브랜드 | 성격 | 법적 근거 | 내부 사업유형 |
|--------|------|-----------|---------------|
| **신속통합기획** | 정비계획 수립 지원 프로그램 | 도시정비법 + 서울시 자체 | 재개발, 재건축 |
| **모아타운** | 소규모주택정비 관리지역 | 소규모주택정비법 제2조9호 | 자율주택, 가로주택, 소규모재건축, 소규모재개발 |
| **모아주택** | 모아타운 내 개별 사업 | 소규모주택정비법 | 가로주택정비사업이 대표적 |

> **핵심**: 모아타운은 `project_type`이 아니라 여러 소규모 사업을 묶는 **관리지역(관리계획)** 개념.
> 모아타운 내 개별 블록은 가로주택/자율주택/소규모재건축/소규모재개발 중 하나로 사업 진행.

## 4. 사업 진행단계 (stage) 표준 순서

```
후보지선정 → 신통자문 → 신통통보 → 신통착수 → 용역착수
→ 추진위 → 주민공람 → 구역지정 → 조합설립 → 시행자지정(신탁)
→ 심의 → 사업시행 → 관리처분 → 착공 → 준공
→ 조합해산 / 해제 (종결)
```

> 신통(신속통합기획) 단계는 서울시 전용. 다른 지자체에는 없음.
