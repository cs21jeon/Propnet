Proppedia - Real Estate Encyclopedia App | Last updated: April 2026

1. About This App

Proppedia is a free mobile app that allows users to look up real estate information in South Korea, including building registers, land registers, and officially assessed prices.

This app retrieves data from Korean government open data APIs using officially issued API authentication keys.

2. Data Sources

All data displayed in this app comes from the following official Korean government sources:

Korea Public Data Portal

URL: https://www.data.go.kr

Operated by the Korean government (Ministry of the Interior and Safety). This is the official Korean government open data platform, similar to data.gov (USA) or data.gov.uk (UK). It provides public APIs for various government datasets.

APIs used by this app:

API Name Provider Data
Building Register API Ministry of Land, Infrastructure and Transport Building information (area, structure, floors, usage, approval date)
Officially Assessed Individual Housing Price API Ministry of Land, Infrastructure and Transport Single-family housing prices
Officially Assessed Apartment Price API Ministry of Land, Infrastructure and Transport Multi-unit housing prices
Officially Assessed Land Price API Ministry of Land, Infrastructure and Transport Individual land prices

VWorld - Spatial Information Open Platform

URL: https://www.vworld.kr

Operated by the Ministry of Land, Infrastructure and Transport (South Korea). VWorld is the official Korean government platform for spatial and land information data.

APIs used by this app:

API Name Data
Land Information API Land category, area, zoning, officially assessed land price
Land Use Plan API Land use regulations and plans
Note on accessibility: VWorld (https://www.vworld.kr) is an official Korean government website that may restrict access from outside South Korea due to regional access policies. The URL is NOT broken — it is fully accessible within South Korea. This app is distributed exclusively in South Korea on Google Play.
