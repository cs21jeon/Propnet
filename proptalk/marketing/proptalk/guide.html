1. 앱 설치

  1. Google Play 스토어에서 "Proptalk"을 검색합니다.
  2. "프롭톡 - 통화녹음 AI요약 업무공유" 앱을 찾아 탭합니다.
  3. 설치 버튼을 눌러 앱을 다운로드합니다.
  4. 설치가 완료되면 앱을 실행합니다.

Play 스토어에서 검색

설치 버튼 탭

TIP: Proptalk은 현재 Android 전용입니다. iOS 버전은 준비 중입니다.

2. 로그인 & 알림 설정

Proptalk은 Google 계정으로 간편하게 로그인합니다. 별도의 회원가입 절차가 없습니다.

  1. 앱을 실행하면 시작 화면이 나타납니다. "Google 계정으로 시작하기" 버튼을 탭합니다.
  2. Google 계정 선택 화면에서 사용할 계정을 선택합니다.
  3. 이용약관과 개인정보처리방침에 동의하면 로그인이 완료됩니다.
  4. 알림 허용 팝업이 나타나면 "허용"을 탭합니다. 채팅방 메시지, 음성 요약 완료 등을 알림으로 받을 수 있습니다.

Google 계정으로 시작하기

계정 선택

알림 허용

TIP: Google 계정으로 로그인하면 Google Drive가 자동으로 연동됩니다. 음성 파일과 요약 결과가 Drive에 자동 백업됩니다.

3. 홈 화면

로그인 후 나타나는 홈 화면에서 Proptalk의 3가지 핵심 기능에 접근할 수 있습니다.

  • 업무 채팅방 가기 - 음성파일 요약과 업무 공유를 편리하게 이용하세요.
  • 음성 요약 보러가기 - 녹음 파일 요약 결과를 확인하세요.
  • 음성 및 업무 파일 확인 - 구글 드라이브에 저장된 원본 음성파일과 업무 파일을 확인하세요.

Proptalk 홈 화면

TIP: 우측 상단의 "창성" 버튼을 탭하면 바로 새 채팅방을 만들 수 있습니다.

4. 업무 채팅방

홈 화면에서 "업무 채팅방 가기"를 탭하면 채팅방 목록으로 이동합니다.

채팅방 목록

참여 중인 채팅방을 한눈에 볼 수 있습니다. 각 채팅방에는 주제, 참여 인원, 역할(관리자/멤버)이 표시됩니다.

채팅방 목록 화면

새 채팅방 만들기

  1. 채팅방 목록 화면 우측 하단의 + 버튼을 탭합니다.
  2. "새 톡방 만들기"를 선택합니다.
  3. 채팅방 이름을 입력하고, 필요시 설명을 추가합니다.
  4. Drive 자동 저장과 시트 기록 옵션을 설정합니다.
  5. "만들기" 버튼을 탭하면 채팅방이 생성됩니다.

새 톡방 만들기 선택

이름 입력 및 옵션 설정

TIP: Drive 자동 저장을 켜면 음성파일이 Google Drive에 자동 백업됩니다. 시트 기록을 켜면 변환 결과가 Google Sheets에 자동 기록됩니다.

초대코드로 참여하기

다른 사람이 만든 채팅방에 참여하려면 초대코드를 사용합니다.

  1. 채팅방 목록 하단의 + 버튼을 탭합니다.
  2. "초대코드로 참여"를 선택합니다.
  3. 전달받은 초대코드를 입력하고 "참여"를 탭합니다.

초대코드로 참여 선택

초대코드 입력

채팅방 대화

채팅방에서 텍스트 메시지를 주고받고, 음성 파일을 업로드하면 AI가 자동으로 요약합니다. 요약 결과는 채팅방에 바로 표시됩니다.

채팅방 대화 화면 - 음성 요약 결과 표시

TIP: 요약 메시지에는 파일 정보(전화번호, 날짜, 녹음 시간)와 함께 "Drive에서 열기" 버튼이 포함됩니다. 탭하면 원본 파일을 바로 들을 수 있습니다.

채팅방 설정

채팅방 상단의 톱니바퀴 아이콘을 탭하면 채팅방 설정을 확인할 수 있습니다.

  • 초대 코드 - 복사 아이콘을 탭해서 팀원에게 공유하세요.
  • Drive 자동 저장 / 시트 기록 - 언제든 켜고 끌 수 있습니다.
  • 멤버 목록 - 참여 중인 멤버와 역할을 확인합니다.
  • 채팅방 나가기 / 채팅방 삭제 - 관리자는 채팅방을 삭제할 수 있습니다.

채팅방 설정 화면

5. 음성 요약

홈 화면에서 "음성 요약 보러가기"를 탭하면, 채팅방별로 정리된 AI 요약 결과를 모아 볼 수 있습니다.

채팅방 선택

음성 요약이 있는 채팅방 목록이 표시됩니다. 확인할 채팅방을 탭합니다.

음성 요약 - 채팅방 목록

요약 카드 목록

채팅방을 선택하면 해당 채팅방의 모든 음성 요약 카드가 시간순으로 표시됩니다. 각 카드에는 제목, 전화번호, 날짜, 녹음 시간, 주제가 표시됩니다.

요약 카드 목록

요약 상세 보기

카드의 펼치기 화살표를 탭하면 AI 요약의 상세 내용을 확인할 수 있습니다. 매물 현황, 고객 상황, 투자 고민, 시장 현실, 결론 등이 구조화되어 표시됩니다.

상단의 편집 아이콘을 탭하면 요약 내용을 수정할 수도 있습니다.

요약 상세 내용 펼치기

요약 검색

상단의 필터 아이콘을 탭하면 요약 내용을 검색할 수 있습니다. 전화번호, 이름, 시작일/종료일로 필터링하여 원하는 요약을 빠르게 찾을 수 있습니다.

검색 필터 화면

음성파일 듣기 (Google Drive 연결)

요약 카드에서 원본 음성을 들으려면 Google Drive 계정을 연결합니다.

  1. 요약 카드의 재생 버튼 또는 Drive 아이콘을 탭합니다.
  2. Google 계정 선택 팝업에서 Drive가 연결된 계정을 선택합니다.
  3. 음성 파일이 로드되어 앱 내에서 바로 재생됩니다.

Google Drive 계정 선택

6. 음성파일 원본 & 오디오 플레이어

홈 화면에서 "음성 및 업무 파일 확인"을 탭하면, Google Drive에 백업된 음성파일 원본을 확인할 수 있습니다.

채팅방별 폴더

채팅방마다 자동 생성된 Drive 폴더가 표시됩니다. Drive 연동 상태도 함께 확인할 수 있습니다.

채팅방별 Drive 폴더 목록

Google Drive 파일 목록

폴더의 외부 링크 아이콘을 탭하면 Google Drive 앱에서 직접 파일 목록을 확인할 수 있습니다. 녹음 파일과 요약 시트가 날짜별로 정리되어 있습니다.

Google Drive 파일 목록

오디오 플레이어

음성파일을 탭하면 앱 내장 오디오 플레이어가 실행됩니다.

  • 재생/일시정지 - 중앙의 재생 버튼으로 조작
  • 10초 앞/뒤로 - 좌우 버튼으로 빠르게 이동
  • 재생 속도 - 1x, 1.5x, 2x 등 속도 조절 가능
  • 자막(CC) - AI 변환 텍스트를 자막처럼 표시
  • 탐색 바 - 원하는 위치로 드래그하여 이동

앱 내장 오디오 플레이어

Google Drive 계정 연결

음성파일 원본에서도 Drive 계정 연결이 필요합니다. 처음 한 번만 계정을 선택하면 이후에는 자동으로 연결됩니다.

Google Drive 계정 선택

TIP: Google Drive에 저장된 파일은 Drive 앱에서도 직접 접근할 수 있습니다. 채팅방 이름으로 폴더가 자동 생성되어 깔끔하게 정리됩니다.

7. 프로필 & 설정

홈 화면 또는 채팅방 목록에서 프로필 아이콘을 탭하면 프로필 화면으로 이동합니다.

앱 설정

  • 다크 모드 - 토글을 켜면 눈이 편안한 다크 테마로 전환됩니다.

법적 문서

  • 이용약관 - 서비스 이용 조건을 확인합니다.

동의 관리

8. PropSheet 연동 중개사 전용

중개사 회원은 PropSheet(매물장)과 Proptalk을 연동하여, 매물장에서 바로 전화 녹음 요약을 확인하고 채팅방 기록을 조회할 수 있습니다.

중개사 전용 기능: 이 기능은 PropSheet(매물장) 서비스에 가입한 중개사 회원만 이용할 수 있습니다.

매물장에서 전화번호 연결

PropSheet의 소유주 연락처 필드에서 전화 아이콘을 탭하면, 해당 번호의 통화 녹음이 Proptalk에 있는지 자동으로 검색됩니다.

매물장에서 전화번호 연결

통화 요약 바로 확인

전화번호를 탭하면 해당 번호의 통화 기록과 AI 요약이 팝업으로 표시됩니다. Drive 파일 링크와 전문 보기도 바로 접근할 수 있습니다.

채팅방 연동 - 녹음 날짜/이름 목록

전화번호와 통화 횟수 기준으로 정리된 목록도 제공됩니다. 각 번호의 요약 내용과 통화 횟수를 빠르게 파악할 수 있습니다.
