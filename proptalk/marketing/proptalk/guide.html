1. 시작하기

Android 앱 설치

  1. Google Play 스토어에서 "Proptalk"을 검색합니다.
  2. 설치 버튼을 눌러 앱을 다운로드합니다.
  3. 설치가 완료되면 앱을 실행합니다.

앱 시작 화면

2. 로그인

Proptalk은 Google 계정으로 간편하게 로그인합니다. 별도의 회원가입 절차가 없습니다.

  1. 시작 화면에서 "Google 계정으로 시작하기" 버튼을 탭합니다.
  2. Google 계정 선택 화면이 나타나면 사용할 계정을 선택합니다.
  3. 이용약관과 개인정보처리방침에 동의하면 로그인이 완료됩니다.
TIP: Google 계정으로 로그인하면 Google Drive가 자동으로 연동됩니다. 음성 파일과 요약 결과가 Drive에 자동 저장됩니다.

3. 채팅방 만들기 & 메시지 공유

로그인 후 메인 화면에서 "업무 채팅방 가기"를 탭하면 채팅방 목록으로 이동합니다.

새 채팅방 만들기

  1. 채팅방 목록 화면 우측 하단의 + 버튼을 탭합니다.
  2. 채팅방 이름과 주제를 입력합니다.
  3. 초대 코드를 공유하면 팀원이 참여할 수 있습니다.

4. 음성 파일 업로드 & AI 요약

Proptalk의 핵심 기능입니다. 통화 녹음 파일을 채팅방에 업로드하면 AI가 자동으로 텍스트 변환 + 핵심 요약을 생성합니다.

음성 파일 업로드

  1. 채팅방 하단의 + 버튼 또는 마이크 아이콘을 탭합니다.
  2. 녹음 파일(m4a, mp3, wav 등)을 선택합니다.
  3. 파일이 업로드되면 AI가 자동으로 텍스트 변환을 시작합니다.
  4. 변환이 완료되면 핵심 요약이 채팅방에 자동으로 표시됩니다.
TIP: 요약 결과에는 주제, 매물 정보, 가격 변경, 결론 등이 마크다운 형식으로 깔끔하게 정리됩니다. 파일명에서 전화번호와 날짜도 자동으로 추출됩니다.

5. 저장된 데이터 확인

업로드한 음성 파일과 AI 요약 결과는 여러 방법으로 확인할 수 있습니다.

음성 요약 보러가기

메인 화면에서 "음성 요약 보러가기"를 탭하면, 채팅방별로 정리된 요약 카드를 확인할 수 있습니다.

  1. 메인 화면에서 "음성 요약 보러가기"를 탭합니다.
  2. 요약을 확인할 채팅방을 선택합니다.
  3. 해당 채팅방의 모든 음성 요약 카드 목록이 표시됩니다.
  4. 카드를 탭하면 상세 요약 내용을 펼쳐 볼 수 있습니다.
