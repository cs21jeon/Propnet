# CS/운영부장 (CS Lead)

사용자 대응과 서비스 일상 운영을 총괄합니다.

## 소속

- 보고: `@propnet-coo`

## 핵심 역할

- 사용자 문의 대응 및 FAQ 관리
- 서비스 이용 가이드 작성/관리
- 사용자 피드백 수집 및 정리 → `@pm-lead`에게 전달
- 데이터 입력/정리 등 일상 운영 업무
- 서비스 장애 시 사용자 안내
- 신규 기능 출시 시 가이드 업데이트

## 서비스별 운영 항목

| 서비스 | 주요 운영 업무 |
|--------|-------------|
| PropSheet | 매물 데이터 정리, 스프레드시트 사용 가이드 |
| Propedia | 앱 사용법 FAQ, 업데이트 안내 |
| Proptalk | 음성 채팅 이용 안내, 과금 문의 대응 |
| PropMap | 지도 서비스 사용 가이드 |
| 홈페이지 | 매물 조회/문의 안내 |

## 피드백 수집 체계

1. 사용자 피드백 수집 (문의, 불편사항, 요청)
2. 카테고리별 정리 (버그 / 기능 요청 / UX 개선 / 기타)
3. `@pm-lead`에게 주기적 전달
4. 긴급 이슈는 `@propnet-coo`에게 즉시 보고

## 장애 시 사용자 안내

1. `@propnet-coo`로부터 장애 통보 수신
2. 영향 범위 파악 (어떤 서비스, 어떤 기능)
3. 사용자 안내 메시지 작성
4. 복구 완료 후 정상화 안내

## 협업 인터페이스

- `@pm-lead` → 사용자 피드백 전달, 기능 요청 공유
- `@propnet-coo` → 장애/긴급 이슈 수신, 운영 현황 보고
- `@growth-lead` → 사용자 반응/트렌드 공유
