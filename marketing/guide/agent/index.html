1. 회원 가입하기

PropNet은 Google 계정 하나로 시작합니다. 회원가입 페이지에서 중개사 등록까지 한 번에 진행할 수 있습니다.

회원유형 선택 & Google 로그인

  1. propnet.kr/register에 접속합니다.
  2. 회원 유형에서 "공인중개사"를 선택합니다.
  3. "Google 계정으로 시작하기" 버튼을 누릅니다.
  4. 사용할 Google 계정을 선택하고 "계속"을 누릅니다.

공인중개사 선택 후 Google 로그인

Google 계정 선택

약관 동의 & 중개사 정보 입력

  1. 서비스 이용 동의 화면에서 "전체 동의"를 체크하고 "동의하고 중개사 등록으로" 버튼을 누릅니다.
  2. 중개사 등록 신청 화면에서 사무소 정보를 입력합니다.
    • 사무소명 — 중개사무소 이름
    • 영문 사무소명 (URL용) — PropSheet 워크스페이스 및 매물지도 주소에 사용됩니다
    • 대표자명 / 연락처
    • 사무소 주소 — 매물지도의 기준 위치로 사용됩니다
    • 중개업 등록번호 / 중개사 등록증 — 등록증 사본(PDF, JPG, PNG)을 업로드합니다
    • 사무소 로고 (선택) — 매물지도에 표시됩니다
  3. "등록 신청하기" 버튼을 누르면 가입 신청이 완료됩니다.

전체 동의 후 진행

중개사 정보 입력

등록증 업로드 후 신청

등록 신청 완료 (승인 1~2일 소요)

TIP: 업무용 Google 계정이 있다면 그 계정으로 가입하세요. 이후 모든 PropNet 서비스에서 같은 계정으로 자동 로그인됩니다.

2. 승인 후 시작하기

가입 후, 관리자가 중개사 등록증을 확인하고 승인해드립니다. 현재는 무료로 시작하실 수 있습니다.

승인 절차

  1. 등록 신청이 완료되면 관리자에게 자동으로 알림이 전달됩니다.
  2. 관리자가 등록증을 확인한 뒤 승인 처리합니다. (보통 1~2일 소요)
  3. 승인이 완료되면 이메일로 안내가 발송되고, 내 워크스페이스가 자동으로 만들어집니다.

승인 완료 시 발송되는 안내 이메일

승인 후 이용 가능한 서비스

  • PropSheet — 매물 데이터를 입력하고 관리하는 스마트 장부
  • PropMap — 내 매물을 지도에 자동으로 표시
  • Propedia — 건축물대장, 공시지가를 30초 만에 조회
  • Proptalk — 통화 녹음을 AI로 요약

3. PropSheet — 매물 데이터 관리

PropSheet는 스마트 매물 장부입니다. 엑셀처럼 매물 정보를 입력하면, 다른 서비스(지도, AI 요약 등)와 자동으로 연결됩니다.

매물 데이터 입력하기

  1. propnet.kr/propsheet에 로그인합니다.
  2. 내 워크스페이스에서 매물 데이터베이스(단일부동산, 부분부동산 등)를 선택합니다.
  3. 상단의 "+ 새 레코드" 버튼을 눌러 새 매물을 추가합니다.
  4. 각 칸(셀)을 클릭하여 정보를 입력합니다.

워크스페이스 — 데이터베이스 선택

스프레드시트 형태의 매물 관리

주요 입력 항목

  • 지번주소 — 매물 위치 (입력하면 지도 좌표가 자동 설정됩니다)
  • 매매가 / 전세가 — 거래 금액
  • 대지면적 / 건물면적 — 면적 정보
  • 대표사진 — 매물 사진 첨부 (클릭 후 파일 선택)
  • 현황(상태) — '등록'으로 변경하면 지도에 공개됩니다
TIP: 주소를 입력하면 지도 좌표가 자동으로 설정됩니다. 직접 좌표를 입력할 필요가 없습니다.

4. PropMap — 내 매물을 지도에 표시

PropSheet에 입력한 매물을 지도 위에 자동으로 표시하는 서비스입니다. 별도의 지도 작업이 필요 없습니다.

매물을 지도에 공개하기

  1. PropSheet에서 공개할 매물의 현황 칸을 클릭합니다.
  2. 드롭다운에서 '등록'을 선택하고 저장합니다.
  3. 곧바로 PropMap 지도에 매물 마커가 나타납니다.

내 매물지도 확인하기

승인 시 발급받은 고유 주소로 나만의 매물지도를 확인할 수 있습니다.

  • 내 지도: propnet.kr/propmap/내부동산 영문이름/
  • 전체 통합 지도: propnet.kr/propmap/

PropMap 매물지도 — 중개사무소 패널과 매물 마커

매물 상세 정보 & 고객 문의

지도에서 매물 마커를 클릭하면 간략 정보가 표시되고, 상세내역보기로 사진과 세부 정보를 확인할 수 있습니다. 고객은 "이 매물 문의하기" 버튼으로 바로 상담을 신청할 수 있습니다.

마커 클릭 — 매물 간략 정보

매물 상세보기 — 사진, 면적, 용도

고객이 보내는 상담 문의 → PropSheet 상담신청 DB에 자동 기록

TIP: 매물을 비공개로 전환하려면 현황을 '등록' 이외의 값으로 변경하세요. 지도에서 즉시 내려갑니다.

5. Propedia — 건축물대장 조회

건축물대장, 토지정보, 공시지가를 30초 만에 조회할 수 있습니다. 현장에서 고객에게 바로 정보를 보여주세요.

건물 정보 검색하기

  1. propnet.kr/proppedia에 접속하거나, 앱을 실행합니다.
  2. 검색 방식을 선택합니다 — 도로명주소, 지번주소, 또는 지도에서 검색.
  3. 검색 결과에서 건물을 선택하면 기본정보, 토지정보 등 상세 정보가 표시됩니다.

검색 방식 선택

조회 결과 — 기본정보, 토지정보

조회 결과 활용하기

  • PropSheet에 저장 — 조회 결과를 바로 PropSheet 매물 데이터로 저장
  • PDF 저장 — 조회 결과를 PDF로 저장하여 고객에게 공유
  • 즐겨찾기 — 자주 조회하는 건물을 저장해두면 빠르게 다시 확인

조회 결과를 PropSheet에 바로 저장

TIP: Google Play에서 "부동산백과"를 검색하면 앱으로도 이용할 수 있습니다. 현장에서 스마트폰으로 바로 조회해보세요.

6. Proptalk — 통화녹음 AI 요약

고객과의 통화 녹음 파일을 올리면, AI가 핵심 내용을 자동으로 정리해줍니다. 메모할 필요 없이 나중에 요약만 확인하세요.

통화 녹음 업로드하기

  1. Google Play에서 "Proptalk"을 설치하거나, 웹 버전에 접속합니다. PC에서는 PWA 앱으로도 설치할 수 있습니다.
  2. Google 계정으로 로그인합니다.
  3. 채팅방을 만들거나 기존 채팅방에 들어갑니다.
  4. 하단의 첨부(클립) 버튼을 눌러 녹음 파일(m4a, mp3, wav)을 선택합니다.
  5. 파일이 업로드되면 AI가 자동으로 텍스트 변환과 요약을 시작합니다.

웹 버전 — 채팅방 목록

PWA 앱 — 채팅방 입장

요약 내용 확인하기

요약이 완료되면 PropSheet 채팅방 데이터베이스에서 녹음 날짜, 전화번호, 요약, 통화내용을 한눈에 확인할 수 있습니다.

PropSheet 채팅방 DB — AI 요약 결과 기록

TIP: 통화요약의 통화정보는 통화녹음파일의 제목을 사용함으로 통신사 또는 제조사의 저장방식에 따라 달라질 수 있습니다.
