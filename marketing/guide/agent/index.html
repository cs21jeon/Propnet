1. 회원 가입하기

PropNet은 Google 계정 하나로 시작합니다. 별도의 가입 양식을 작성할 필요가 없습니다.

  1. propnet.kr/propsheet에 접속합니다.
  2. "Google 계정으로 로그인" 버튼을 누릅니다.
  3. 사용할 Google 계정을 선택합니다.
  4. 이용약관과 개인정보처리방침에 동의하면 가입이 완료됩니다.

2. 승인 후 시작하기

가입 후, 관리자가 중개사 회원으로 승인해드립니다. 현재는 무료로 시작하실 수 있습니다.

승인 절차

  1. 가입이 완료되면 관리자에게 자동으로 알림이 전달됩니다.
  2. 관리자가 중개사 정보를 확인한 뒤 승인 처리합니다.
  3. 승인이 완료되면 내 워크스페이스가 자동으로 만들어집니다.
빠른 시작: 승인을 빨리 받고 싶으시면 가입 후 아래 연락처로 문의해 주세요.
전화/카카오톡: 010-XXXX-XXXX

승인 후 이용 가능한 서비스

  • PropSheet — 매물 데이터를 입력하고 관리하는 스마트 장부
  • PropMap — 내 매물을 지도에 자동으로 표시
  • Propedia — 건축물대장, 공시지가를 30초 만에 조회
  • Proptalk — 통화 녹음을 AI로 요약

3. PropSheet — 매물 데이터 관리

PropSheet는 스마트 매물 장부입니다. 엑셀처럼 매물 정보를 입력하면, 다른 서비스(지도, AI 요약 등)와 자동으로 연결됩니다.

매물 데이터 입력하기

  1. propnet.kr/propsheet에 로그인합니다.
  2. 내 워크스페이스에서 매물 데이터베이스를 엽니다.
  3. 하단의 + 버튼을 눌러 새 매물을 추가합니다.
  4. 각 칸(셀)을 클릭하여 정보를 입력합니다.

4. PropMap — 내 매물을 지도에 표시

PropSheet에 입력한 매물을 지도 위에 자동으로 표시하는 서비스입니다. 별도의 지도 작업이 필요 없습니다.

매물을 지도에 공개하기

  1. PropSheet에서 공개할 매물의 상태 칸을 클릭합니다.
  2. 드롭다운에서 '등록'을 선택합니다.
  3. 저장하면 곧바로 PropMap 지도에 매물이 나타납니다.

5. Propedia — 건축물대장 조회

건축물대장, 공시지가를 30초 만에 조회할 수 있습니다. 현장에서 고객에게 바로 정보를 보여주세요.

건물 정보 검색하기

  1. propnet.kr/proppedia에 접속하거나, 앱을 실행합니다.
  2. 검색창에 도로명 주소 또는 지번 주소를 입력합니다.
  3. 검색 결과에서 건물을 선택하면 상세 정보가 표시됩니다.

6. Proptalk — 통화녹음 AI 요약

고객과의 통화 녹음 파일을 올리면, AI가 핵심 내용을 자동으로 정리해줍니다. 메모할 필요 없이 나중에 요약만 확인하세요.

통화 녹음 업로드하기

  1. Google Play에서 "Proptalk"을 설치하거나, 웹 버전에 접속합니다.
  2. Google 계정으로 로그인합니다.
  3. 채팅방을 만들거나 기존 채팅방에 들어갑니다.
  4. 하단의 + 버튼을 눌러 녹음 파일(m4a, mp3, wav)을 선택합니다.
  5. 파일이 업로드되면 AI가 자동으로 텍스트 변환과 요약을 시작합니다.
